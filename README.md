# Kraken Clone - React & Tailwind CSS

This repository contains a high-fidelity clone of the [Kraken](https://www.kraken.com/) landing, business, and sign-up pages. Originally built in pure HTML/CSS, the application has been entirely re-architected into a scalable Single Page Application (SPA).

## Technologies Used
- **React.js (Vite)** - For blazing-fast frontend rendering and component-based architecture.
- **Tailwind CSS v4** - For pixel-perfect custom styling, complex responsive layouts, and granular theme controls.
- **React Router DOM** - For seamless, client-side routing between the Home, Business, and Signup pages.

## Features Replicated
- Exact color palettes, typography, and UI aesthetics.
- Precise hover animations, glassmorphism, and gradients mapped perfectly to Tailwind utility classes.
- Exact custom SVG illustrations and videos.

## Getting Started

To run the project locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Build the project for production:

```bash
npm run build
```
